## Get Started Locally
1. Create a virtual env by your choice:
```bash
conda create --prefix ./env python=3.11
conda activate ./env
```
2. Install the requirements:
```bash
pip install -r requirements.txt
```
3. Set up your .env variables (fal.ai API):
```bash
FAL_KEY=xxxxxxxx
```
4. Run the app:
```bash
python app.py
```
